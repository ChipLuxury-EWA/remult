## Nuxt Example

This is a [Nuxt](https://nuxt.com/) based Todo app that uses Remult and [Sidebase nuxt-auth](https://github.com/sidebase/nuxt-auth) for authentication.

### Running locally

```bash
npx degit remult/remult/examples/nuxt-todo remult-nuxt-todo
cd remult-nuxt-todo
npm install
npm run dev
```
